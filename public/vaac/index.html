Visual Acuity Analysis & Compensation

A modern digital platform to analyze your vision, detect color blindness, and get instant suggestions — all from the comfort of your screen.

Start Vision Test → Color Blindness Test

What You Can Do

Three powerful tools, one simple interface.

👁️

Vision Test

Check your visual acuity with letters, numbers, and shapes that gradually decrease in size and contrast.

🎨

Color Blindness

15 Ishihara-style color plates to detect normal vision, red-green deficiency, or partial color blindness.

📄

Instant Report

View score, accuracy, suggestions and download a PDF report — saved in your browser history.

About this Project

This system was built as a final-year project to demonstrate how web technologies can support preventive eye-care. It combines vision-acuity testing with color-blindness screening in a single, easy-to-use web application — making basic eye-health awareness accessible to everyone.
